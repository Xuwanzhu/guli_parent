# guli_parent
这是一个《在线教育学习平台》的后端代码。。

项目为前后端分离开发，后端采用SpringCloud微服务架构，持久层用的是MyBatis-Plus，
微服务分库设计，使用Swagger生成接口文档。接入了阿里云视频点播、阿里云OSS。

系统分为前台用户系统和后台管理系统两部分。
前台用户系统包括：首页、课程、名师、问答、文章。
后台管理系统包括：讲师管理、课程分类管理、课程管理、统计分析、Banner管理、订单管理、权限管理、问答管理、文章管理等功能。
